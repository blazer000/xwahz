期刊信息

《新闻爱好者》全国新闻核心期刊是河南省一级期刊。河南日报报业集团主办。以广博性、可读性、实用性为宗旨，关注新闻、追踪热点、传递信息、普及知识。《新闻爱好者》自创刊以来，先后获河南省优秀期刊提名奖、优秀栏目设置奖，被评为省一级期刊，全国第二届、第三届、第四届、第五届新闻核心期刊等。2005年度又被龙源期刊网评为中文期刊网络阅读亚洲排行第63名。

本刊份上半月（大众版）主要栏目有：新闻与传播研究、传媒观察、新闻三味、网络传播、随笔、媒体经营、新闻大学、报海钩沉、等。下半月（理论版），栏目设置：新闻与传播研究、传媒观察、现代广告、媒体经营网络传播、新闻大学、新闻与法治、报海钩沉等。

编稿流程

编辑审稿

首先由编辑进行信息的核对已经初审工作，编辑完成工作后提交至编辑部初审分配。

专家审稿

由编辑部负责稿件的初审工作，在严谨的审查之后，通过初审的稿件提交至专家进行专家审稿阶段。

稿件复审

专家审稿完毕后，交由总编进行复审，在严谨的审查之后，通过复审的稿件会提交至编辑部进行定稿。

主编定稿

定稿之后，本刊编辑部会进行邮件以及电话通知发稿人，并在期刊发行之后邮寄样刊。
